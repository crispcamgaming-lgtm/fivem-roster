🚀 Getting Started — Setup Checklist
Admin Only
✓
Connected to Supabase
Database connection is active. Your data is live and synced.
✓
Add max_slots Column to fudt_ranks (v10 — one-time)
Run this in your Supabase SQL Editor to enable rank slot limits:
ALTER TABLE fudt_ranks ADD COLUMN IF NOT EXISTS max_slots integer DEFAULT 0;
2
Create Departments
Go to Admin → Departments and add each department (e.g. SAHP, BCSO, LSPD). Give each a name, abbreviation, and color.
3
Add Ranks for Each Department
Go to Admin → Ranks, select a department, and add all ranks in order (1 = highest). Assign each a tier (HC, LC, Patrol, etc.).
4
Connect Google Sheets (Optional)
Go to Admin → Sheets. Paste your Google Sheet URL for each department and enter the exact tab name. Your Supabase Edge Function handles the proxy.
5
Map Sheet Column Headers
Go to Admin → Field Mapping. For each department with a sheet, type the exact column headers from your spreadsheet into each field. Use Auto-Detect to speed this up.
6
Create User Accounts & Assign Roles
Go to Admin → User Accounts to create accounts for HC members. Assign them the hc role and their department. Members can self-register and you approve them under Pending.
7
Set Up Discord Webhooks (Optional)
HC members can configure Discord webhooks under the HC Panel → Discord Webhook tab. This posts LOA approvals, rank changes, and more to your Discord channels automatically.
8
Run Your First Sync
Once sheets and field mapping are configured, click the ▶ Sync button in the top bar or go to Admin → Sync Log → Force Sync. The system auto-syncs every 30 minutes.

TOOLKIT GUIDE
Overview · Commands · Authorization
🤖 What is this Toolkit?

This universal toolkit is designed to eliminate admin paperwork across all departments. It centralizes roster management, discipline logging, subdivision tracking, and more — so you spend more time on duty than behind a desk.

While being authorized with the toolkit is not required, it is highly recommended as it makes your life a whole lot easier.
🔑 Getting Authorized
1
Create a Setup Ticket
Head to your department's ticket channel and open a Toolkit Setup ticket.
2
Register Your Account
Click "Create one here" on the login screen. An Admin or HC will approve and assign your role.
3
You're Good to Go
All commands are locked to a permission system. If something doesn't work, open a ticket.
⚠️ Important Notes
⛔
You physically cannot use the toolkit without completing authorization steps above.
⚠️
Do not spam commands. If something doesn't work once, report it — don't retry.
ℹ️
All commands are permission-gated. If you think you should have access to a command, make a ticket.
